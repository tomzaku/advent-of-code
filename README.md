# Haskell

To get the output

```sh
cat day-1.in | runhaskell day-1.hs
```

To save on the cache
```sh
cat day-1.in | runhaskell day-1.hs | pbcopy
```

To save on files
```
cat day-1.in | runhaskell day-1.hs > day-1.out
```
